### Установка шаблона:
    1. Скачиваем шаблон
    2. В папке, куда скачали шаблон открываем терминал
    3. Пишем `cp BT\ Clean\ Module.xctemplate ~/Library/Developer/Xcode/Templates`
    4. Теперь при создании нового файла есть выбор нашего шаблона
